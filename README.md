# SoundSketch 🎵

SoundSketchは、音楽クリエイターがふと浮かんだメロディ、ビート、コード進行などのアイデアを素早く録音してストックするための、ミニマルで洗練された音声メモアプリケーションです。

## ✨ 主な機能

- **🎙️ ワンタップ録音**: 大きな録音ボタンで、アイデアを逃さずすぐに録音可能。
- **⏱️ 内蔵メトロノーム**: Web Audio API を活用した、高精度でズレのない録音用メトロノーム（40〜240 BPM）。録音中も動作します。
- **💾 データの永続化保存**: IndexedDB を使用し、ブラウザを閉じたりリロードしたりしても録音データが失われません。
- **🌐 高音質エクスポート**: 録音した音声を標準的な Wav ファイル（16-bit PCM）として直接ダウンロードできます。ブラウザでの非互換性を防ぐフォールバック機能も搭載。
- **🌊 オーディオビジュアライザ**: 録音中および再生中に、音の大きさに応じてリアルタイムで動くスタイリッシュな波形（ビジュアライザ）を表示します。
- **📝 メモとタイトル編集**: 録音したデータに、テキストでメモ（キー、BPM、コード進行など）を追加可能。タイトルもタップするだけで直接編集できます。
- **🏷️ カラフルなタグ機能**: 
  - 「メロディ」「ビート」「コード」などの候補タグをワンタップで追加。
  - 独自のカスタムタグも作成可能。自動的に割り当てられるカラーで視覚的に整理できます。
- **🔍 絞り込み検索と並び替え**: 追加したタグでのフィルタリング（複数タグのAND検索）や、「新しい順」「古い順」「長い順」など柔軟な並び替えに対応しています。
- **📱 PWA＆スマホ最適化**: iPhone 等のスマートフォンから「ホーム画面に追加」することで、ネイティブアプリのようにフルスクリーンで快適に動作します。画面下部に固定された操作パネルにより、片手での操作もスムーズです。

## 🛠️ 使用技術

- **フロントエンド**: React 18, Vite
- **スタイリング**: Tailwind CSS v4
- **アイコン**: Lucide React
- **音声録音・処理**: MediaRecorder API, Web Audio API (AnalyserNode, OscillatorNode)
- **データ永続化**: IndexedDB (idb-keyval)

## 🚀 ローカルでのセットアップと実行

このプロジェクトをローカル環境で動かすための手順です。
前提条件として、[Node.js](https://nodejs.org/) がインストールされている必要があります。

1. リポジトリのクローン（またはプロジェクトフォルダへの移動）
2. 依存関係のインストール
   ```bash
   npm install
   ```
3. 開発サーバーの起動
   ```bash
   npm run dev
   ```
   ※ `vite.config.js` にて `host: true` に設定されているため、同じWi-Fiネットワーク内にあるスマートフォン等のブラウザから `http://<PCのローカルIP>:5173/` にアクセスして実機テストを行うことができます。
4. ブラウザでアクセス
   ターミナルに表示されたローカルURL（例: `http://localhost:5173/`）をブラウザで開いてください。

> ⚠️ **注意**: 
> 録音機能（マイクへのアクセス）を使用するため、ブラウザのマイク許可設定を有効にしてください。
